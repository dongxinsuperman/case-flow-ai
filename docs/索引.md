# 公开文档索引

本页只列出随开源仓库发布的文档。

## 第一次使用

1. 阅读 [README](../README.md) 了解定位和最小运行方式。
2. 阅读 [功能说明](功能说明.md) 确认每项能力需要哪些配置。
3. 按 [开发与运维](开发与运维.md) 启动 PostgreSQL、后端和前端。
4. 不接飞书时可以直接使用 [快速模式](快速模式.md)。

- 新增公开文档：加入本索引。
